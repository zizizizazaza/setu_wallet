1. 产品定位：基于谷歌浏览器的插件钱包
2. 用户landing路径：
    1. 插件安装，点击进入landing page
        1.1 创建钱包:获得助记词，确认助记词
        1.2 导入钱包：导入助记词
    2. 创建密码，确认密码
    3. 完成，进入钱包插件页面
3. 插件需求：
    1. 主页：展示总资产余额，展示token / activity 
         token列表：展示token余额(当前仅FLUX)
         activity列表：展示交易历史activity
    2. 账户管理
        点击账户，可查看所有账户/钱包
        选中单个账户即回到首页、并切换账户
        点击账户的更多选项，跳转至账户详情页
            账户详情页展示账户地址、可修改备注昵称、可查看私钥、可删除账户、可查看助记词

    3. 切换网络
        默认仅支持cetu主网；支持自定义添加网络
    4. 设置
        支持修改密码、设置语言
        查看服务条款、隐私政策
        展示github、x
